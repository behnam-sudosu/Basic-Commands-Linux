# Permission

1.
## file
### r w x 
### r ===>> read
### w ===>> write
### x ===>> excute
    chmod u+w (NameFile) ===>> u = user, add w to file
    chmod u-w (NameFile) ===>> take w to file
    chmod g+wx (NameFile) ===>> g = group
    chmod o+rwx (FileName) ===>> o = others
    chmod -r or -R ===>> all file in folder
    chmod 777 (FileName)
